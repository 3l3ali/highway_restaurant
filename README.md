# 🍕 About  
A Web App for Highway Restaurants Chain in Jordan, Customers Could Browse Their Menus, Place Orders For Delivery and Check Restaurant's Locations.

# 🔥 Technologies  
- Ruby On Rails
- Html
- SCSS
- Bootstrap
- Responsive Design
- JavaScript
- Google Maps Api
- PostgreSQL

# 🌎 Hosting  
The Web App is Hosted on Heroku.
